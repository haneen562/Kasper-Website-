# 🎨 Kasper — Portfolio Website

A personal portfolio website built from scratch with HTML and CSS, showcasing projects, skills, and contact info.

## 🔗 Live Demo
[View Live](https://haneen562.github.io/Kasper-Website-)

## ✨ Features
- Smooth scroll navigation across sections
- Projects showcase with live demo links
- Skills and experience sections
- Fully responsive across mobile, tablet, and desktop
- Custom CSS animations and hover effects

## 🛠️ Tech Stack
- HTML5
- CSS3 (Flexbox, Animations)


## 🚀 How to Run
1. Clone the repo: `git clone https://github.com/haneen562/Kasper-Website-`
2. Open `index.html` in your browser
